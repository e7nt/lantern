# v0.1.3 release candidate — 2026-07-21

## Release outcome

`v0.1.3` is a bounded Apple Silicon developer-preview release. It adds the
persistent Git-aware workbench explorer and removes balanced inline Markdown
chrome from agent answers at the terminal presentation boundary.

The verified `v0.1.2` candidate built successfully on Apple Silicon, but its
tagged workflow was cancelled before publication when Intel packaging was
paused. The immutable tag was not moved or reused; this corrected boundary uses
the next patch version.

This checkpoint does not claim completion of Lantern's full product roadmap.
Guided Build playback, structured learning missions and change narratives,
multi-folder workbenches, Live Collaboration, Intel and Linux release artifacts
remain outside this patch release.
